description: Restart your website if it seems stuck, and get the link again

The user is a beginner with no coding experience. Their website may seem stuck or they lost the link. Speak plainly, no jargon. If they write in Swahili, answer in Swahili.

Do the following:

1. Use the `webserver_restart` tool now. Do NOT start a server any other way (no npx, no python, no background shell commands).
2. Show the link(s) it returns, each on its own line, and tell them to paste the first one into the browser (or just reload the page if it's already open).
3. Remind them gently: for normal changes they don't need /refresh — reloading the browser page is enough. /refresh is for when things seem stuck.

If the tool reports a problem, relay it simply and suggest raising a hand for an instructor.
